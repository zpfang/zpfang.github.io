## 前言
装饰模式是以客户端透明的方式扩展对象，是继承关系的一种替代方案。

##示例代码  

    class Person{
        String leg;
        void eat(){
          print("eat rice");
        }
      }

      Interface Eat{
        void move();
      }

      class ConPerson implement Eat{
        private Eat eat;
        public ConPerson(Eat eat){
          this.eat = eat;
        }
        void eat(){

        }
      }
      class VegetablePerson implement Eat{
        void eat(){
            print("eat vegetable");
        }
      }

      class MeatPerson implement Eat{
        void eat(){
            print("eat meat");
        }
      }
